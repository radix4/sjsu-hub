# SJSU Hub


### Dependencies
  - Front-end
    - React
    - React Bootstrap
    - React Dom
    - React Scripts
    - StockJS Client
    - Web

  - Back-end
   - Java 11
   - Maven Project
   - Spring Web
   - Spring Data JPA
   - MySQL Driver
   - WebSocket (SockJS and STOMP)
   - Lombok


### How to Run
  - Front-end
   - sjsu-hub\frontend> npm install
   - sjsu-hub\frontend> npm start
   
  - Back-end
   - sjsu-hub\backend> ./mvnw spring-boot:run
